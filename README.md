# springboot_integrated
springboot整合其他框架的案例

整合了
Freemarker,
Thymeleaf,
Mybatis,
Servlet,
Filter,
Listener,
Jsp,
DevTools,
Spring Data Redis,
Spring Data JPA,
Spring task定时任务,
Quartz,
单元测试,
Ehcache,
文件上传,
对表单数据校验,
异常处理,
全局配置文件的多环境配置，
自定义日志logback的输出，
使用actuator监控Spring Boot的健康状况,
解决了IDEA中使用DevTools热部署不生效的问题
